## 新手建议

**第一次使用？** 建议按顺序阅读：

1. 🎲 **生成随机激励** → 了解基础随机化
2. 📊 **收集覆盖率** → 理解验证完整性
3. 🗄️ **创建寄存器模型** → 掌握DUT配置
4. 📋 **完整流程** → 看所有模块如何协同

**熟悉SystemVerilog？** 直接查看：
- [SV→Python概念映射](../concepts/sv-to-python-mapping.md)
